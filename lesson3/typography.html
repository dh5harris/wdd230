Typography

Heading Font:

We will use the font "Marcellus SC" from Google Fonts. Marcellus SC works well for headings. It gives a good visual impact with some elegance, and is easy to read.

Paragragph Font:

We will use the font "Open Sans" from Google Fonts. Open Sans is an easy to read font on all scenes. It works well the main text of the website.
